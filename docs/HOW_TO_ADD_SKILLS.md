# Как добавить скиллы (Skills) в Antigravity

Скиллы — это наборы инструкций, скриптов и ресурсов, которые расширяют возможности ИИ-агента. Они позволяют агенту выполнять специализированные задачи, используя проверенные методики.

## Структура скилла

Каждый скилл должен находиться в своей собственной папке. Минимальный набор файлов:

- `SKILL.md` (обязательно): Основной файл с инструкциями и метаданными.
- `scripts/` (опционально): Вспомогательные скрипты.
- `examples/` (опционально): Примеры использования.

### Формат SKILL.md

Файл должен начинаться с YAML фронтматтера:

```markdown
---
name: имя-скилла
description: Краткое описание того, когда использовать этот скилл.
---

# Инструкции для скилла
Здесь описываются шаги, которые агент должен выполнить.
```

## Куда устанавливать скиллы

Вы можете установить скиллы в два места:

1. **Для конкретного проекта**: `.agent/skills/` внутри корневой папки проекта (`c:\Proj\ProjIB_VS\.agent\skills/`).
2. **Глобально**: `~/.gemini/antigravity/skills/` (обычно в `C:\Users\i_max\.gemini\antigravity\skills\`).

## Скиллы от сообщества (Community Skills)

Вы можете использовать готовые скиллы от сообщества. Вот несколько популярных репозиториев:

*   **[rominirani/antigravity-skills](https://github.com/rominirani/antigravity-skills)**: Коллекция примеров и туториалов.
*   **[guanyang/antigravity-skills](https://github.com/guanyang/antigravity-skills)**: Скиллы для разработки, планирования и мультимедиа.
*   **[heilcheng/awesome-agent-skills](https://github.com/heilcheng/awesome-agent-skills)**: Курируемый список инструментов и скиллов для ИИ-агентов.

### Как установить скилл из сообщества:

1. Найдите нужный скилл в репозитории (например, папку `git-commit`).
2. Скопируйте эту папку в ваш каталог `.agent/skills/`.
3. Перезапустите сессию агента.

**Совет**: Если вы дадите мне ссылку на конкретный скилл на GitHub, я могу попытаться скачать и установить его для вас автоматически.

## Создание собственного скилла

Используйте шаблон в `.agent/skills/template` как основу для создания своих собственных инструментов.
